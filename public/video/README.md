# Hero Video

Place the hero video files here:

- `hero.webm` — WebM format (preferred, ~30% smaller)
- `hero.mp4` — MP4 fallback

## Requirements

- Resolution: 1920x1080 minimum
- Duration: 15-30 seconds (looped)
- Content: Bali drone shot, yoga sunrise, or tropical nature
- Max file size: 15MB per file
- Encoding: H.264 for MP4, VP9 for WebM

## Free Sources

- [Pexels](https://pexels.com/videos) — Search "bali drone", "yoga sunrise"
- [Coverr](https://coverr.co) — Search "tropical nature cinematic"

## Usage

Once video files are added here, update `Hero.tsx` to use a `<video>` element
instead of the static poster image with zoom animation.
